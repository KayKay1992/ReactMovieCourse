USECONTEXT HOOK

3. useContext
The useContext hook allows you to access the value of a context directly within your components. Context is useful for passing data through the component tree without having to manually pass props at each level.

Example:
jsx
Copy
import React, { useContext } from 'react';

// Create a context
const ThemeContext = React.createContext('light');

function ThemedComponent() {
  const theme = useContext(ThemeContext); // Access the current context value
  return <div>The current theme is {theme}</div>;
}

function App() {
  return (
    <ThemeContext.Provider value="dark">
      <ThemedComponent />
    </ThemeContext.Provider>
  );
}

export default App;
The useContext hook reads the theme from ThemeContext, which is provided by the ThemeContext.Provider.
